## 🛠️ How to Run Each Project

### For Express.js Projects
```bash
cd Express
npm install
npm start
```

### For MVC Application
```bash
cd MVC
npm install
npm start
```

### For URL Shortener
```bash
cd shortUrl
npm install
npm start
```

### For Simple Node.js Projects
```bash
cd HELLO
node hello.js
# or
node math.js
```

### For File Operations
```bash
cd FILE
node file.js
```

### For Server Implementation
```bash
cd SERVER
node index.js
```

### For Practice Projects
```bash
cd project1

cd project2
node index.js
```
